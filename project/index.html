Похитители космоса

Правила игры:

  • Главная цель игры набрать как можно больше очков, очки начисляются за уничтожение врага. За каждого врага начисляются 100 очков.
  • На момент старта у игрока есть 3 жизни, жизнь расходуется при попадании врага в главного героя либо при попадании игрока в астеройд.
  • Управление игроком осуществляется нажатием стрелок на клавиатуре "влево" / "вправо".
  • Стрелять во врагов нужно клавишей "пробел".
